Software

Dr. Scratch website

plataforma web permite subir programas de Scratch, un lenguaje de programación visual orientado al aprendizaje de técnicas de programación para niños, para ser evaluados. El alumno puede ganar medallas y desarrollar sus habilidades de programación y alcanzar niveles de mayor complejidad y calidad. Dentro del proyecto se realizarán actividades específicamente orientadas al fomento de la programación para chicas, colectivos desfavorecidos y discapacitados.

LibreSoft Gymkana website

an open source educational and geolocated game related to M-Learning and Tourism applications. It is based on the LibreGeoSocial mobile social network application, which includes a mobile augmented reality interface. You can organize and manage a mobile gymkana using Android smartphones without great effort.
